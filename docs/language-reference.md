# Language Reference

`tnix` keeps Nix value syntax and adds type-only syntax for checking and tooling.

This document is the reference entry point for the language surface used in
`.tnix` and `.d.tnix`.

## Scope

The reference covers:

- file kinds
- expression forms
- annotations and declarations
- type constructors, aliases, and indexed containers
- numeric validation and unit types
- inference notes, erasure, and common usage patterns
